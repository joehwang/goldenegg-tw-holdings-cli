# 台灣證交所即時報價 API 規格

## API 端點
```
https://mis.twse.com.tw/stock/api/getStockInfo.jsp?ex_ch={exchange}_{symbol}.tw&json=1&delay=0
```

## 參數說明
- `exchange`: 交易所代碼
  - `tse`: 台灣證券交易所（上市）
  - `otc`: 櫃買中心（上櫃）
- `symbol`: 股票代號（如 2330, 6488）
- `json=1`: 回傳 JSON 格式
- `delay=0`: 即時資料

## MarketDataProvider 使用方式

### 1. 取得完整股票資訊
```python
from models.market_data import MarketDataProvider

provider = MarketDataProvider()
info = provider.get_stock_info("2330")

print(f"股票名稱: {info['name']}")
print(f"最新成交價: {info['current_price']}")
print(f"今日最高價: {info['high_price']}")  # ✅ 有最高價
print(f"今日最低價: {info['low_price']}")   # ✅ 有最低價
print(f"漲跌幅: {info['change_percent']:.2f}%")
print(f"成交量: {info['volume']} 張")
```

### 2. 只取收盤價（向後相容）
```python
price = provider.get_closing_price("2330")
```

### 3. 批量查詢
```python
prices = provider.get_multiple_closing_prices(["2330", "2317", "6488"])
```

## 支援功能

✅ **有漲跌幅**: 自動計算 `(當前價格 - 昨收價) / 昨收價 * 100`  
✅ **有最高最低價**: 欄位 `h` (最高) 和 `l` (最低)  
✅ **有成交量**: 欄位 `v`  
✅ **有漲跌停價**: 欄位 `u` (漲停) 和 `w` (跌停)  
✅ **支援上市上櫃**: 自動偵測 TSE/OTC  
✅ **五檔委買委賣**: 欄位 `a`, `b`, `f`, `g`  

## 資料更新頻率
- 盤中：即時更新（延遲約數秒）
- 盤後：顯示收盤資料
- 休市：顯示前一交易日資料
